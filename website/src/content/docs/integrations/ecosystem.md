Ecosystem integrations focus on distribution and interoperability beyond a single framework.

## Priority packages

- **MCP Server (`@graphrefly/mcp-server`)**: universal adapter surface for MCP-capable agent clients.
- **AI SDK middleware (`@graphrefly/ai-sdk`)**: inject reactive state and policy-aware flow into AI SDK pipelines.
- **LangGraph tools (`@graphrefly/langgraph`)**: connect GraphReFly capabilities into LangGraph-centered stacks.

## Why this category exists

These packages reduce adoption friction by meeting users where they already are, while keeping GraphReFly as the coordination substrate.
