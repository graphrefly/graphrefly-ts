Core Concepts

Built on first principles.

◈

Reactive Precision

Combining Callbag-ish minimalism with Diamond-safe correctness. A protocol small enough for an LLM to reason about, yet robust enough to eliminate race conditions through a rigorous two-phase push mechanism (DIRTY then DATA). Every flow is glitch-free, high-performance, and mathematically sound.

♦

Governed Integrity

Trust is not an add-on; it is the protocol. With ABAC (Attribute-Based Access Control) and Versioning baked directly into both nodes and graphs, every state change is authorized, immutable, and auditable. Complex reactive systems remain stable, secure, and effortlessly reversible.

◼

Living Blueprints

Moving from ephemeral code to Introspectable Artifacts. Graphs are modular, pluggable solutions that encapsulate Values, Status, and Algorithm into a single, serializable unit. Through the Companion Store, every step is accountable and every metadata point is inspectable.

☆

Agentic Co-operation

A symbiotic workspace where Humans master the abstraction and AI masters the implementation. By unifying security, observability, and composability, GraphReFly provides the perfect arena for agents—human or LLM—to play to their strengths and rely on a shared, transparent logic.

One protocol. Every spectrum.

A single message format — tuples of [Type, Data?] — powers sources, sinks, operators, and entire graphs. The same primitive adapts to every use case, from real-time UI bindings to LLM agentic memory.

Developer Experience

Clean, expressive, typed.

Sugar constructors keep the common path simple. The protocol stays accessible underneath.

graph.ts 
const count   = state(0);
const doubled = derived([count], (c) => c * 2);

effect([doubled], (d) => console.log(d));

count.down([[DATA, 1]]);  // logs: 2
