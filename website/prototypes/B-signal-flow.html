── CORE PROPERTIES ──
[01]
One Protocol

Sources, transforms, and sinks are all nodes. Every node speaks the same message tuple protocol: [Type, Data?]. No special cases, no type hierarchies.

[02]
Diamond-Safe

Two-phase push (DIRTY then DATA) prevents glitches in diamond-shaped dependency graphs. Nodes resolve only after all upstream paths deliver.

[03]
Graph as Artifact

The Graph container holds topology, snapshot, and describe metadata. Inspect or serialize the full execution state at any point. Observability without interference.

[04]
Human + LLM

Designed for co-operation. LLMs observe the graph, propose mutations via messages, and receive feedback through the same protocol humans use. Shared ground truth.
