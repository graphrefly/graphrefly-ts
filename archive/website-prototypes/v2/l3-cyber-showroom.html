Three things state
doesn't solve.

01 / Audit

Every value has a trail.

describe({ explain }) walks backwards through dependencies and returns a structured chain. Humans audit it. Agents reason over it. No black boxes.

Read the explain spec
02 / Projection

Spec is a projection of code.

factoryTag-stamped factories carry self-description into the live graph. describe() emits the topology as JSON. The blueprint is never out of date.

See the projection model
03 / Co-op

Subgraphs have owners.

L0 static, L1 TTL, L2 heartbeat, L3 supervisor — the ownership staircase lets multi-agent worktrees edit a shared topology without corrupting it.

Read the ownership protocol
