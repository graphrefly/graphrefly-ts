[PILLAR 1] priority: critical
>>> STOP DROWNING IN INFORMATION

Universal Reduction Engine

Hundreds of emails, alerts, feeds, logs — every day. GraphReFly's reactive graph continuously reduces raw streams into the few items that actually matter, without polling and without you writing the plumbing.

[PILLAR 2] priority: critical
>>> TRUST AI BY UNDERSTANDING IT

Traceable Decisions & Causal Chains

Every node checkpoints its state. Every derivation records its causal chain. When an LLM recommends an action, you can walk the graph backward and see exactly which inputs led to that conclusion.

[PILLAR 3] priority: high
>>> DESCRIBE IT, DON'T CODE IT

NL → Graph → Run

Tell the system what you need in plain language. The LLM composes the reactive graph — nodes, edges, effects — and the runtime executes it. No boilerplate. No framework lock-in. Just intent to action.

~ ~ ~ ~ ~ ~ ~ ~ ~ ~ ~ ~ ~ ~ ~ ~ ~ ~
user@graphrefly:~$ graphrefly --pipeline
# Step 1: Describe your intent
echo "Triage inbox, flag VIP deadlines" \
# Step 2: LLM composes the reactive graph
| graphrefly compose --llm gpt-4o \
# Step 3: Run reactively with checkpoints and traces
| graphrefly run --checkpoint --trace
# Output: only the items that need your attention,
# with a full causal trace of why.
