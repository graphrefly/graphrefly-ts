▲ Elevation Profile

Three layers of terrain

L.01
▲

Protocol Layer

Messages as tuples — [Type, Data?]. A minimal, universal wire format. Two-phase push (DIRTY then DATA) ensures diamond-safe propagation with zero glitches.

L.02
▲

Node Layer

Stateful primitives that compose freely. State, derived, and effect nodes wire up via the protocol. Every node is observable without disturbing data flow.

L.03
▲

Graph Layer

The container that makes the topology inspectable. Describe, snapshot, replay. Graphs become persistent artifacts both humans and LLMs can reason over.

▲ Elevation Markers

Built for the terrain ahead

▲ N01

One protocol

Every node speaks the same tuple messages. Sources, sinks, operators — all compose through a single handshake. No special cases.

▲ N02

Diamond-safe

Two-phase DIRTY/DATA push eliminates glitch artifacts in diamond-shaped graphs. Derived values are always consistent, never stale.

▲ N03

Graph as artifact

Describe, snapshot, serialize. The graph is not hidden runtime state — it is a first-class inspectable artifact that persists across sessions.

▲ N04

Human + LLM

Observability without disturbing data flow. Both humans and language models can inspect, modify, and reason over the same reactive graph.
