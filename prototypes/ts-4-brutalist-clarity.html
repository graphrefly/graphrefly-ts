How it works

Three steps.
No magic.

01

Describe

Tell the system what you want in plain language. "Triage my inbox. Flag urgent items. Summarize the rest into a daily digest. Archive noise." That is your entire spec.

02

Compose

An LLM reads your description and emits a reactive graph. Source nodes for ingestion. Derived nodes for classification. Effect nodes for output. All wired together declaratively.

03

Run

The graph runs persistently. State checkpoints automatically. Every derivation is traced. Inspect, override, or extend any node at runtime. Reactive propagation, not polling.

Code

Familiar
primitives.

Unfamiliar power.

counter.ts TypeScript 
import { Graph } from '@graphrefly/graphrefly';
import { state, derived, effect } from '@graphrefly/graphrefly/core';
import { DATA } from '@graphrefly/graphrefly/protocol';

// 1. Create a reactive graph
const g = new Graph();

// 2. State node — holds a value, pushes updates
const count = g.add(state(0));

// 3. Derived node — recomputes when count changes
const doubled = g.add(derived(
  [count],
  (c) => c * 2
));

// 4. Effect node — side-effect on every change
g.add(effect(
  [doubled],
  (d) => console.log(`doubled: ${d}`)
));

// 5. Push an update — entire graph reacts
count.down([[DATA, 1]]);
// logs: doubled: 2
